# Penny

Penny is a personal expense tracker that helps users manage their income, expenses, budgets, and spending insights. It provides monthly dashboards, transaction history, budgeting tools, and analytics to help users better understand their financial habits.

## Functional Requirements

- [ ] User authentication using Clerk
- [ ] Add income transactions
- [ ] Add expense transactions
- [ ] Each transaction includes:
      - Amount
      - Date
      - Category
      - Optional note

- [ ] View transaction history
- [ ] Update transactions
- [ ] Delete transactions

- [ ] Filter transactions by:
      - Type (Income / Expense)
      - Category
      - Date range

- [ ] Search transactions by category or note

- [ ] Dashboard displaying:
      - Total income
      - Total expenses
      - Net cash flow
      - Spending by category (Pie Chart)
      - Income vs Expense (Bar Chart)
      - Current month's financial summary

- [ ] Monthly budget management
      - Set budget
      - Edit budget
      - Delete budget
      - Budget by category

- [ ] Category management
      - Add category
      - Rename category
      - Delete category

- [ ] Analytics page
      - Highest spending category
      - Average daily expense
      - Month-over-month spending change
      - Best saving month
      - Savings rate
      - AI-generated financial insights

- [ ] Analytics time filters
      - Past month
      - Past 3 months
      - Past 6 months
      - Past year

- [ ] Responsive UI

- [ ] Light and Dark theme
